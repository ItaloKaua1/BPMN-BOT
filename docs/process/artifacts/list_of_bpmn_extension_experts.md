# When Should Experts Be Consulted?

Experts should be consulted when:

* The extension introduces new BPMN constructs.
* The integration with BPMN elements is unclear.
* Conflicts between concepts are identified.
* The metamodel requires validation.
* Concrete syntax decisions need confirmation.
* The extension requires external evaluation.

---

# Expected Contributions

Domain and BPMN experts may contribute by:

* Reviewing extension concepts.
